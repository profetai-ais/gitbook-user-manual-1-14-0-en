## Display Language

<figure><img src="../.gitbook/assets/image (110).png" alt=""><figcaption></figcaption></figure>

1. Click the "Preferences" icon in the right sidebar.
2. On the Preferences page, click the Supported Languages dropdown menu.
3. Under the "Display Language" option, select the interface language to display (e.g., English / Traditional Chinese / Simplified Chinese / Japanese / Thai).

> Note: In the Thai interface, Agent and Token are currently still displayed in English.
