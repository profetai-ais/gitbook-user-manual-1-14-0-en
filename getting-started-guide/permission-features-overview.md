### **Dataset**

By assigning users, groups, or organizations to a dataset, administrators can control which users can use that dataset in other features (such as agent configuration, testing, or other knowledge-based features).

After saving permission settings, the selected users, groups, or organizations will be able to access and use that dataset in supported features.
