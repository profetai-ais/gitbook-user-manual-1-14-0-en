# Workflow Template Management

## Introduction

Workflow templates are used with the workflow assistant, allowing users to visually design how prompts entered by users during conversations with the workflow assistant are handled.

> **Note**: By default, only AI Studio administrators can modify these settings.

## **Create New Workflow Template**

> For workflow editing operations, see the instructions in [Edit Workflow](../agent-zhi-nan/bian-ji-gong-zuo-liu-cheng.md)

## **Ways to Create a New Template**

#### **Blank Template**

This option creates a blank workflow template.

<figure><img src="../.gitbook/assets/image (249).png" alt=""><figcaption></figcaption></figure>

1. Click the "Add" button in the upper right corner
2. In the "Name" field, enter the name, then click the button on the right to create multi-language labels. See [Multi-language Settings](gong-zuo-liu-cheng-mu-ban-guan-li.md#duo-guo-yu-yan-she-ding)
3. In the "Description" field, enter the description, then click the button on the right to create multi-language labels. See [Multi-language Settings](gong-zuo-liu-cheng-mu-ban-guan-li.md#duo-guo-yu-yan-she-ding)
4. Click the "OK" button to confirm. The newly created workflow will appear in the template list. Click the name to open the editing view.

#### Multi-language Settings

<figure><img src="../.gitbook/assets/image (250).png" alt=""><figcaption></figcaption></figure>

1. Click the "globe" button on the screen for automatic translation. Users can also manually edit the content.
2. After automatic translation is complete, click the "OK" button to save the content.

> Note: The large language model options in the "Model" menu depend on the configuration of your actual installation environment. The options shown in this documentation are for reference only.

#### **Copy from Template**

This option allows users to copy an existing workflow template into a new workflow for editing.

<figure><img src="../.gitbook/assets/image (252).png" alt=""><figcaption></figcaption></figure>

1. Select the template to copy from the template list
2. Click the Copy button
3. The newly created workflow will appear in the template list

#### **Import Template**

This option allows users to import a workflow template from a file into a new workflow for editing.

<figure><img src="../.gitbook/assets/image (253).png" alt=""><figcaption></figcaption></figure>

1. Click the more button in the upper right and select Import File
2. Click the "File Upload Area" to select the template file to import (files with the `.pwflow` file extension)
3. In the "Name" field, enter the name, then click the button on the right to create multi-language labels. See [Multi-language Settings](gong-zuo-liu-cheng-mu-ban-guan-li.md#duo-guo-yu-yan-she-ding)
4. In the "Description" field, enter the description, then click the button on the right to create multi-language labels. See [Multi-language Settings](gong-zuo-liu-cheng-mu-ban-guan-li.md#duo-guo-yu-yan-she-ding)
5. Click the "OK" button again to complete the addition. The newly created workflow will appear in the template list.
