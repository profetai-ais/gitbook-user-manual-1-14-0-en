  An Agent is a conversational application that allows users to enter text (prompts), which are processed through scenario-specific mechanisms and sent to a Large Language Model (LLM) to generate responses.
---

# Agent Overview

## **Key Advantages**

<figure><img src="../.gitbook/assets/image (122).png" alt=""><figcaption></figcaption></figure>

* **Versatility**: Offers multiple Agent types to meet different application needs. High-quality Agent applications can be created with simple configuration.
* **Governance**: Agent usage permissions can be managed to ensure generative AI applications comply with enterprise policies.
* **Accuracy**: Combines information from knowledge bases to generate responses, reducing the risk of hallucination.
* **Flexibility**: Supports hybrid application scenarios; developers can swap LLM models/services at any time based on actual needs.
