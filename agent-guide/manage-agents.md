# Manage Agents

## **Agent Types**

<figure><img src="../.gitbook/assets/image (136).png" alt=""><figcaption></figcaption></figure>

* **Chat Agent**: Directly interacts with internal or external large language models through basic Q&A.
* **Workflow**: Can orchestrate workflows to handle complex application scenarios.

> Note: A "Chat Agent" cannot search internal enterprise information to generate responses. To use internal enterprise knowledge, use a "Knowledge" type Agent.

## **Assign Agent Library Administrators**

<figure><img src="../.gitbook/assets/image (137).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (138).png" alt=""><figcaption></figcaption></figure>

1. Click the "... (More)" button at the top left of the screen
2. Select "Assign"
3. In the popup, click the "User Menu" to select a user, or type to filter
4. Click the **"Agent Library Role Menu"** to select the role to assign to the user
5. Click the "Invite" button to complete the permission assignment
6. Click the "Remove" button at the far right of a user row to revoke that user's permissions

> To change a user's assigned role, click the "Agent Library Role Name" in that user's row to reselect

## Export / Import Agent (JSON)

The export/import feature can be used to back up Agent configurations, move settings between environments, or quickly duplicate similar Agents to reduce setup effort.

### Export Agent

<figure><img src="../.gitbook/assets/image (139).png" alt=""><figcaption></figcaption></figure>

1. Go to the Agent list, or enter the edit page of the target Agent.
2. Hover your mouse over the Agent row you want to export.
3. Click the Export button — the system will download a proas file.

### Import Agent

<figure><img src="../.gitbook/assets/image (140).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (141).png" alt=""><figcaption></figcaption></figure>

1. Click the "... (More)" button at the top left of the screen
2. Select "Import File"
3. Click the field to select the file to import
4. Click "OK" to import the specified file
